# wled-backup
Backups of my WLED device configurations. Generated and kept up to date by [wled-manager](https://github.com/bscholer/wled-manager)

## Naming convention
| Prefix | Meaning     |
| ------ | ----------- |
| bl-    | Backlight Lamp |
| cl-    | Corner Lamp |
| dl-    | Desk Lamp   |
| ml-    | Matrix Lamp |
| x-     | exterior    |
